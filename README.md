# 游戏存档位置说明

常见（Windows）存档位置：
1. Steam（本地）  
   `...\Steam\userdata\<Steam 好友代码>\<AppID>\remote`
2. 文档目录  
   `%USERPROFILE%\Documents\(<开发团队/公司/发行商名>\)<游戏名>\`
   或 `%OneDrive%\文档\(<开发团队/公司/发行商名>\)<游戏名>\`
3. 本地应用数据（常见）  
   `%USERPROFILE%\AppData\Local\<游戏名>\`  
   或 `%USERPROFILE%\AppData\LocalLow\(<开发团队/公司/发行商名>\)<游戏名>\`
4. Roaming 配置  
   `%USERPROFILE%\AppData\Roaming\<游戏名>\`

如何查找 AppID：
- AppID：Steam 商店页面 URL 中的数字，或在 https://steamdb.info/ 搜索游戏名。也可以在游戏属性 -> 更新中查看。
