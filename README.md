# 游戏存档位置说明

常见（Windows）存档位置：
1. Steam（本地）  
   `D:\Steam\userdata\<Steam 好友代码>\<AppID>\remote`
2. 文档目录  
   C:\Users\<用户名>\Documents\<游戏名>\
3. 本地应用数据（常见）  
   C:\Users\<用户名>\AppData\Local\<游戏名>\  
   或 C:\Users\<用户名>\AppData\LocalLow\<游戏名>\
4. 漂亮的地方（少数游戏）  
   C:\Program Files (x86)\<游戏目录>\ 或游戏安装目录
5. Roaming 配置  
   C:\Users\<用户名>\AppData\Roaming\<游戏名>\

如何查找 AppID：
- AppID：Steam 商店页面 URL 中的数字，或在 https://steamdb.info/ 搜索游戏名。也可以在游戏属性 -> 更新中查看。
